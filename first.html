So...there are some important points to keep in mind

  1. First of all, choose a programming language and download a code editor.
  2. Along with this, start exploring various domains. Do you want to become a website developer or perhaps you would prefer to be able to build mobile apps or work in software engineering? You will need to think carefully about this because it will largely dictate the languages you are going to learn
  3. Learning 10 programming languages is not necessary but learning one programming language well and doing great in data structures and algorithms is necessary.
  4. Doing 1000 coding problems is not necessary but solving 300 quality problems covering all the topics is necessary.
  5. Ability to solve every new problem is unnecessary but having a learning attitute to learn in new techniques and revise those concepts is necessary.
  6. Starting early gives you an edge always!
  7. Never rush into learning.Your equation should be practice more than you learn.
  8. Your attitude towards learning is extremely important. Always keep a positive attitude and be patient.
  9. No need to run towards competitive programming now, first clear you basics and have some hands-on practice on DSA questions.
  10. Last but not the least, nobody is great. Everybody is learning!

Just keep these points in mind and then nobody can stop you from becoming a great programmer!
